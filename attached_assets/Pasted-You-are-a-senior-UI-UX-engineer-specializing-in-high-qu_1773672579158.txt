You are a senior UI/UX engineer specializing in high-quality real estate rental platforms similar to Zillow Rentals and Apartments.com.

Your task is to upgrade the property listing cards across the entire website to a premium, modern rental platform design while keeping the existing functionality and property data structure intact.

IMPORTANT RULES

• Do NOT break existing property data loading.  
• Do NOT modify backend logic or database structure.  
• Do NOT change routing or property detail page links.  
• Only improve the UI design, structure, and interactive behavior of the property cards.  
• The design must remain responsive for desktop, tablet, and mobile.

OBJECTIVE

Transform the existing property cards into a modern, high-quality rental listing card design with strong visual hierarchy and smooth interactions similar to top rental platforms.

PROPERTY CARD STRUCTURE

The card must follow this visual order:

1. Property Image Area  
2. Price  
3. Property Details (Beds • Baths • Sqft)  
4. Address  

Do NOT include landlord or agent information on the card.  
Landlord information must remain only on the property detail page.

IMAGE THUMBNAIL

• The property image must be the main visual element of the card.  
• Use a consistent aspect ratio of 4:3.  
• The image must span the full width of the card.  
• Images should be high resolution and properly cropped.  
• Add a smooth hover zoom effect.

Example hover behavior:

transform: scale(1.05)

PHOTO COUNT INDICATOR

Display the number of available property photos.

Position:
Bottom-right corner of the image.

Example:
📷 12 Photos

Style:
• small semi-transparent dark background  
• white text  
• rounded pill shape  
• subtle padding  

SAVE / FAVORITE BUTTON

Add a heart icon in the top-right corner of the image.

Behavior:
• clicking toggles the saved state  
• hover highlight effect  
• must not break if backend save functionality does not exist  

LISTING BADGES (RENTAL PLATFORM ONLY)

Since this website is focused only on rental listings, do NOT include “FOR SALE”.

Instead support these badges:

FEATURED  
NEW LISTING  
AVAILABLE NOW  
VERIFIED  
HOT DEAL  

Badge placement:
Top-left corner of the image.

Badge style:
• white background  
• small rounded pill shape  
• subtle shadow  
• bold small text  

PRICE DISPLAY

The price must be the most visually prominent text.

Example:
$2,350 / month

Style guidelines:

• large font size  
• bold weight  
• strong contrast color  

PROPERTY DETAILS ROW

Display property specifications in a compact format.

Example:
2 Beds • 2 Baths • 1,200 sqft

Icons may optionally be used for better readability.

ADDRESS DISPLAY

Display the property address below the property details.

Example:
245 Madison Ave  
New York, NY

Address text should be slightly muted to avoid competing with the price.

CARD STYLING

Use a modern, premium card style.

border-radius: 14px

Internal padding:
16px

Default shadow:
0 2px 8px rgba(0,0,0,0.08)

Hover shadow:
0 10px 25px rgba(0,0,0,0.15)

Spacing must remain consistent across all cards.

HOVER INTERACTIONS

When the user hovers over a property card:

• image slightly zooms  
• card shadow becomes deeper  
• cursor becomes pointer  

These effects must feel smooth and modern.

OPTIONAL IMAGE PREVIEW

If multiple images exist for a property, enable image preview cycling when the user hovers over the card.

Behavior example:

Hover → image changes every 1.5 seconds.

Fallback:
If the listing has only one image, display the image normally.

PERFORMANCE OPTIMIZATION

• enable lazy loading for images  
• prefer WebP format when available  
• prevent layout shifts when images load  

RESPONSIVE GRID

Ensure property cards display correctly in a responsive grid.

Desktop:
4 cards per row

Tablet:
2 cards per row

Mobile:
1 card per row

FINAL REQUIREMENTS

After completing the upgrade:

• all cards must still link correctly to the property detail page  
• existing property data must render normally  
• card spacing must remain consistent  
• layout must remain responsive  

The final result should look like a premium modern rental platform with clean visual hierarchy and smooth interactions.
